## 🎯 Descripción

Landing page para presentar HabitFlow. Desplegada en Vercel.

## ✨ Secciones

- Hero con CTA
- Características
- Capturas de pantalla
- FAQ
- Footer

## 📁 Estructura

```
habitflow-landing/
├── app/
│   ├── layout.tsx
│   └── page.tsx
├── components/
│   ├── Hero.tsx
│   └── Features.tsx
└── public/
```

## 🚀 Instalación

```bash
npm install
npm run dev      # Desarrollo
npm run build    # Producción
```
