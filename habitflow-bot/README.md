# HabitFlow Bot 🤖

Bot de Telegram para registro rápido de hábitos y gastos.

![Python](https://img.shields.io/badge/Python-3.11-3776AB?logo=python)
![Telegram](https://img.shields.io/badge/Telegram-Bot-26A5E4?logo=telegram)

## 🎯 Descripción

Bot de Telegram para registrar hábitos y gastos sin abrir la app.

## 💬 Comandos

| Comando | Descripción |
|---------|-------------|
| `/start` | Vincular cuenta |
| `/habits` | Ver hábitos de hoy |
| `/done <nombre>` | Marcar completado |
| `/expense <cantidad> <categoría>` | Registrar gasto |
| `/stats` | Ver estadísticas |

## 📁 Estructura

```
habitflow-bot/
├── bot.py
├── handlers/
│   ├── habits.py
│   └── expenses.py
└── requirements.txt
```

## 🚀 Instalación

```bash
pip install -r requirements.txt
python bot.py
```
